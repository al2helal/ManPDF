\hyperdef{}{NAME}{\section{\hyperref[NAME]{NAME}}\label{NAME}}

atof - convert a string to a double

\hyperdef{}{SYNOPSIS}{\section{\hyperref[SYNOPSIS]{SYNOPSIS}}\label{SYNOPSIS}}

\begin{verbatim}
#include <stdlib.h>
 
double atof(const char *nptr);
\end{verbatim}

\hyperdef{}{DESCRIPTION}{\section{\hyperref[DESCRIPTION]{DESCRIPTION}}\label{DESCRIPTION}}

The \textbf{atof}() function converts the initial portion of the string
pointed to by \emph{nptr} to \emph{double}. The behavior is the same as

~

strtod(nptr, NULL); \\

~

except that \textbf{atof}() does not detect errors.

VALUE}}\label{RETURNux5fVALUE}}

The converted value.
